- **Architected a high-performance Full-Stack application**, utilizing **React, TypeScript, Tailwind CSS, and Vite** for a highly visual frontend, alongside an **Express/Node.js** backend utilizing WebSockets for sub-second real-time threat alerts.

- **Designed intuitive dashboard analytics** using **ApexCharts** to deliver a global threat index and sentiment history, translating complex AI insights into accessible UI components for end-users.
- **Implemented secure user workflows** utilizing JSON Web Tokens (JWT) and Bcrypt, managing customizable keyword tracking, real-time alert preferences, and GDPR-compliant data handling.

![YT-DLP-UI](https://raw.githubusercontent.com/Shawshank01/yt-downloader-electron/main/public/YT-Downloader-UI.png)

This is a modern, user-friendly desktop Electron application for downloading videos from YouTube and other supported platforms. It uses yt-dlp for download, and FFmpeg to change the format or add subtitles and thumbnails.

[Proxy_sh](https://github.com/Shawshank01/proxy_sh)

This is an automated shell script that effortlessly installs and manages Docker containers on GNU/Linux systems. It can be used to construct encrypted and obfuscated traffic proxy servers using Xray (VLESS-XHTTP-REALITY) and Shadowsocks (2022). Supports IPv6.

[xAI-native](https://github.com/Shawshank01/xAI-native)

This is a local, user-friendly web application that provides a graphical interface for chatting with xAI's latest models via their official API. This UI provides full Markdown-rendered streaming responses with context, and image generation capabilities.
